# Real Light

It's not only a defi app, but also a lightweight web app framework.

Just clone it and control it, dont be controlled.

## Features

- [x] Modular
- [x] Livereload
- [x] Pug
- [x] Data Binding
  - [x] Vue2
- [ ] Router
- [ ] Boundle
- [ ] Preprocess
  - [ ] Less
  - [ ] Stylus
  - [ ] Typescript

## Develop

```
npm install
npm run dev
```

Then open `http://localhost:35729/main?env=dev` in your browser.

## Release

```
npm run build [dev|test|live]
```
